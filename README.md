# 请答题小程序

这是由西安电子科技大学请答题团队开发维护的请答题小程序

技术栈：taro + react hooks + typescirpt

组件库：taro ui + vant weapp （更推荐自己造轮子）

## 开始

```bash
// 克隆仓库
git clone git@github.com:ascodelife/taro-qingdati.git
// 拉取依赖
yarn
// 打包成微信小程序文件并监听文件改动
yarn dev:weapp
```
